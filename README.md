Project Overview
  This project is a personal profile web page created using pure HTML5.
  The goal of the project is to understand and apply semantic HTML elements, proper document structure, and accessibility best practices.

What I Have Done
  Created a project folder and an index.html file following real-world naming conventions
  Written the HTML boilerplate manually (DOCTYPE, html, head, body)
  Used semantic HTML elements such as header, nav, section, article, and footer to structure the page
  Added personal details including name, bio, education, skills, and contact information using appropriate HTML tags
  Included a profile image with a meaningful alt attribute for accessibility
  Implemented internal navigation using anchor links to jump between section
  Added comments in the code to explain the purpose of semantic elements
  Tested the page in a web browser to ensure correct rendering

Technologies Used
  HTML5
